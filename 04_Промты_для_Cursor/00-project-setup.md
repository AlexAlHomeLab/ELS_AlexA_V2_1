# Промт: Настройка проекта ELS AlexA V2.1


Создать структуру проекта для токарного станка с ЧПУ на Arduino Mega 2560.


1. Создать все папки и файлы согласно структуре
2. Настроить конфигурацию для Arduino Mega
3. Определить базовые макросы и типы


- src/core/hal/ - аппаратный слой
- src/core/motion/ - планировщик и генерация шагов
- src/core/process/ - шпиндель, СОЖ, E-Stop
- src/core/ui/ - дисплей, кнопки, энкодеры
- src/core/fsm/ - конечные автоматы
- src/core/debug/ - отладка
- src/modes/ - режимы работы
- src/config/ - конфигурация


- 16 МГц
- 256 КБ Flash
- 8 КБ SRAM
- 4 КБ EEPROM
- 54 цифровых пина
- 16 аналоговых входов
- 6 таймеров








- Кнопки и энкодеры работают
- Дисплей показывает меню
- Настройки сохраняются в EEPROM
- Отладка в Serial


Создайте структуру проекта и базовые файлы конфигурации. Начните с hal_pins.h, config.h и главного файла ELS_AlexA_V2.1.ino.
