# Промт: Асинхронная подача


Реализовать режим асинхронной подачи.


1. Ручное управление джойстиком
2. РГИ (ручной генератор импульсов)
3. Установка лимитов
4. Циклы ext/int




- Обработка ручного управления
- Подачи по осям X и Z
- Скорость от потенциометра
- Джойстик и РГИ


- Внешнее точение (ext)
- Внутреннее точение (int)
- Параметры: глубина, проходы
- Циклическое выполнение


- Джойстик: влево/вправо/вверх/вниз
- РГИ: точное позиционирование
- Rapid: ускоренная подача
- Лимиты: установка/сброс


- Скорость подачи: 0-1000 мм/мин
- Глубина резания: 0.1-5 мм
- Количество проходов: 1-99


1. Врезание на глубину
2. Точение по оси
3. Отвод резца
4. Возврат на ускоренной подаче
5. Повтор с уменьшением счетчика


Реализуйте режим асинхронной подачи с ручным управлением и циклами.
