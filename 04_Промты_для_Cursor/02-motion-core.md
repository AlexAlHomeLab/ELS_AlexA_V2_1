# Промт: Motion Core


Реализовать ядро управления движениями.


1. Планировщик движений (look-ahead)
2. Трапециевидная акселерация
3. Генерация шагов по прерываниям (DDS)
4. Компенсация люфта
5. Лимиты и защита




- Алгоритм обратного времени (GRBL)
- Планирование до 16 блоков
- Соединение блоков с оптимальной скоростью
- Атомарные операции с буфером


- DDS генератор шагов
- Прерывания таймера до 30 кГц
- Логарифмическая арифметика для скорости
- Максимально быстрые операции


- Компенсация люфта по осям X и Z
- Выборка при смене направления
- Настраиваемая дистанция
- Отключаемая опция


- Программные лимиты
- Контроль положения
- Защита от выхода за пределы


- Максимальная скорость: 5000 мм/мин
- Ускорение: 500 мм/с²
- Разрешение: 0.01 мм
- Частота шагов: до 30 кГц


planner_add_move(X, 100.0, 50.0); // движение по X на 100 мм со скоростью 50 мм/мин
stepper_generate_steps(); // генерация шагов в прерывании
