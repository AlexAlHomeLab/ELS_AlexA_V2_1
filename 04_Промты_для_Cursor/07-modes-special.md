# Промт: Специальные режимы


Реализовать специальные режимы: фаска, конус, сфера, делительная головка, шлифовка.




- Фаска: снятие кромки
- Угол: 0-90 градусов
- Автоматический расчёт
- Циклы ext/int


- Конус левый/правый
- Параметры: угол, длина
- Многопроходная стратегия


- Сфера: радиус, глубина
- Интерполяция дуг (из Marlin)
- Торцевая обработка


- Делительная головка
- Количество частей
- Отображение угла
- Сброс в 0


- Подача в тысячных
- Ограничения скорости для безопасности
- Управление подачей СОЖ


// Фаска
chamfer_start(angle=45, depth=1.0, passes=3);

// Конус
cone_start(angle=30, length=50.0, type=CONE_LEFT);

// Делительная головка
divider_set_parts(36); // 36 частей по 10 градусов


Реализуйте все специальные режимы с единым API и безопасностью.
