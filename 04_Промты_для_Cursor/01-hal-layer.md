2. Макросы для быстрых операций с портами
3. Настройка таймеров и прерываний
4. Поддержка разных платформ




- Определение всех пинов

- Прерывания таймеров


- Внешние прерывания для энкодеров

// Настройка таймера
void timer1_init(uint16_t period) {
TCCR1A = 0;
TCCR1B = (1 << WGM12) | (1 << CS10);
OCR1A = period;
TIMSK1 |= (1 << OCIE1A);
}
