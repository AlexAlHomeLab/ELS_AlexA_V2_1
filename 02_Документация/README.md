ELS AlexA V2.1

Электронная гитара для токарного станка на базе Arduino Mega 2560.

Описание

ELS AlexA V2.1 - это система ЧПУ для токарного станка, реализующая:
- Асинхронную и синхронную подачу
- Нарезание резьбы по ГОСТ
- Обработку фасок, конусов, сфер
- Делительную головку
- Шлифовку

Аппаратная платформа

- Arduino Mega 2560
- LCD дисплей 2004
- Шаговые двигатели с драйверами
- Энкодер шпинделя (3000 имп/об)
- Ручной генератор импульсов (РГИ)
- Джойстик подач
- Потенциометр скорости

Структура проекта

ELS_AlexA_V2.1/
- src/
- core/       - Ядро системы
- modes/      - Режимы работы
- config/     - Конфигурация
- docs/         - Документация
- scripts/      - Вспомогательные скрипты

Быстрый старт

1. Открой ELS_AlexA_V2.1.ino в Arduino IDE
2. Выбери плату: Arduino Mega 2560
3. Загрузи прошивку
4. Подключи аппаратуру по схеме
