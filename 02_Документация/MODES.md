Режимы работы ELS AlexA V2.1

Список режимов

0 - Асинхронная подача (ручное управление, циклы ext/int)
1 - Синхронная подача (синхронизация с энкодером шпинделя)
2 - Фаска (снятие фаски под углом)
3 - Нарезание резьбы (резьба по ГОСТ, многопроходная)
4 - Конус левый (обработка конуса слева направо)
5 - Конус правый (обработка конуса справа налево)
6 - Делительная головка (деление окружности на части)
7 - Сфера (обработка сферических поверхностей)
8 - Шлифовка (точная шлифовка в тысячных)

Подрежимы

Каждый режим имеет три подрежима:
- MAN - ручной, перемещение джойстиком и РГИ
- EXT - внешнее точение, автоматический цикл
- INT - внутреннее точение, автоматический цикл

Управление

Джойстик:
- MAN: перемещение по осям
- EXT/INT: запуск/остановка цикла

РГИ (ручной генератор импульсов):
- Точное позиционирование
- Выбор оси: X или Z
- Разрешение: 1:1 или 0.01 мм

Кнопки под дисплеем:
- LEFT/RIGHT: навигация по меню
- UP/DOWN: изменение параметров
- SELECT: выбор/подтверждение
- SELECT (долгое): вход в настройки

Потенциометр: установка скорости подачи
Rapid (ускоренная подача): включение максимальной скорости

Циклы EXT/INT

Асинхронная подача (mode_async)

EXT (внешнее точение):
1. Врезание на глубину
2. Точение по оси
3. Отвод резца
4. Возврат на ускоренной подаче
5. Повтор с уменьшением счетчика

INT (внутреннее точение):
1. Врезание на глубину (внутрь)
2. Точение по оси
3. Отвод резца (наружу)
4. Возврат на ускоренной подаче
5. Повтор с уменьшением счетчика

Нарезание резьбы (mode_thread)

Особенности:
- Синхронизация с фазой шпинделя
- Многопроходная стратегия
- Таблицы резьб по ГОСТ
- Дистанции разгона и торможения

Параметры:
- Шаг резьбы (из таблицы)
- Длина резьбы
- Количество проходов (автоматически)

Фаска (mode_chamfer)
Параметры: угол (0-90°), глубина съёма, количество циклов

Конус (mode_cone)
Параметры: угол конуса, длина конуса

Делительная головка (mode_divider)
Параметры: количество частей (2-360), отображение текущего угла

Сфера (mode_sphere)
Параметры: радиус, глубина

Шлифовка (mode_grind)
Особенности: шаг изменения в тысячных, ограниченная скорость, управление подачей СОЖ
