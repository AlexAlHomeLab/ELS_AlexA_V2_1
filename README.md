📌 О проекте
ELS AlexA V2.1 — это система электронного управления подачами (Electronic Lead Screw) для токарного станка на базе Arduino Mega 2560.

Проект реализует полный спектр функций управления движениями суппорта, синхронизацию с вращением шпинделя и автоматические циклы обработки для различных операций точения.

💡 О разработке
Весь код проекта написан в среде Cursor AI — интеллектуальной IDE с глубокой интеграцией AI-ассистентов. Автор проекта постарался учесть лучшие практики и проверенные алгоритмы из таких проектов, как:
GRBL — планировщик движений с look-ahead
Marlin — интерполяция дуг и компенсация люфта
LinuxCNC — синхронизация со шпинделем и циклы резьбы
FastAccelStepper — высокоскоростная DDS-генерация шагов
EncButton — надёжная обработка кнопок и энкодеров

🚀 Быстрый старт
1. Установка
bash
git clone https://github.com/yourusername/ELS_AlexA_V2.1.git
cd ELS_AlexA_V2.1
2. Установка библиотек
Требуемые библиотеки (установить через менеджер библиотек Arduino IDE или PlatformIO):

EncButton — v3.7.3+

hd44780 — для LCD дисплея

GyverIO — зависимость EncButton

3. Сборка и загрузка
Откройте ELS_AlexA_V2.1.ino в Arduino IDE

Выберите плату: Arduino Mega 2560

Нажмите Загрузить

4. Настройка параметров
Все настройки хранятся в файле src/config/config.h:

cpp
#define STEPS_PER_MM_X    200.0f   // Шагов на мм для оси X
#define STEPS_PER_MM_Z    200.0f   // Шагов на мм для оси Z
#define MAX_SPEED_X       5000.0f  // Макс. скорость X (мм/мин)
#define MAX_SPEED_Z       5000.0f  // Макс. скорость Z (мм/мин)
#define ENABLE_BACKLASH   1        // Включить компенсацию люфта
#define BACKLASH_X        10       // Люфт X (импульсы)
#define BACKLASH_Z        10       // Люфт Z (импульсы)
