# Навык: Code Review


Автоматическая проверка кода на соответствие стандартам.




- Все строковые константы должны быть в PROGMEM
- Использовать макрос F() для Serial вывода


- Все критические секции защищены
- Разделяемые данные только в ATOMIC_BLOCK


- Все указатели проверяются на NULL
- Используются assert() для критических параметров


- Все циклы проверяют границы
- Индексы не выходят за пределы


- Используется switch вместо if-else цепочек
- Все состояния обработаны


- Проверка условий в начале функции
- Быстрый выход при ошибках


- camelCase для переменных
- UPPER_CASE для констант и макросов
- snake_case для функций
- PascalCase для типов


1. Сканирование кода на нарушения
2. Отчёт с указанием строк
3. Рекомендации по исправлению
